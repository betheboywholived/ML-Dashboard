# ML Dashboard

A Streamlit-based dashboard for experimenting with machine learning models (classification and regression).

## Setup

```bash
# Create and activate a virtual environment (optional but recommended)
python -m venv .venv
# On Windows
.venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

## Run the app

```bash
streamlit run app/app.py
```

The main entry point is `app/app.py` and the pages live under `app/pages/`.
